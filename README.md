# ContactBook

## Gradle project

To deploy app engine project
```gradle appengineDeploy```

To build gradle
```gradle build```

To clean gradle
```gradle clean```

To redownload the dependencies
```gradle --refresh-dependencies```


## Google cloud commands:

```gcloud auth list ```    

```gcloud auth login```


To set default project 

```gcloud config set project {projected}```
